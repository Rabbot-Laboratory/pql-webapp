# RABBOT LABORATORY UI リデザイン

ご提示いただいた「RABBOT LABORATORY」のロゴ（歯車があしらわれたメカニカルな黒白のウサギ）は非常にクールで、インダストリアル・サイバーな雰囲を持っています！
このロゴの「トンマナ（トーン＆マナー）」に合わせて、現在の「柔らかく暖かみのあるベージュ系UI」から、**ハイテクで専門的なロボット制御コンソール（High-end Cyber / Dark Mode）**へと一新させる提案です。

## User Review Required

> [!IMPORTANT]
> 根本的なデザイン・カラーパレットの変更となるため、いくつかの方針からお好みのものを選んでいただく必要があります。内容をご確認いただきフィードバックをお願いします。

## 提案するデザインコンセプト（トンマナ）

### 案A: 「Cyber Dark」スタイル（おすすめ）
ロゴの「黒白のメカニカルさ」を際立たせるため、画面全体を**ダークモード（深みのある黒）**に変更します。

* **背景色**: 深い黒（Deep Black / #0B0C10）
* **パネル色**: 半透明のダークグレー、またはガラス風の黒（Dark Glass / #1F2833）
* **アクセントカラー**: テクニカルな印象を与える「ネオンシアン（水色）」またはロボティクスらしい「メカニカルオレンジ」
* **UIの形状**: 現在の丸っこい角（radius）を削り、よりシャープで工業的なエッジ（四角形に近い角丸）に変更

*ロゴは白黒反転（CSSフィルタ等）させてダーク背景に最も映える形で左上に配置します。*

### 案B: 「Clean Lab（クリーン・ラボ）」スタイル
現在の明るい画面を維持しつつ、ベージュなどの暖色を排除し、「白・グレー・黒」の無彩色によるミニマルな研究室スタイルにします。

* **背景色**: 純白（Pure White / #FFFFFF）とライトグレー
* **アクセントカラー**: 洗練された「モノトーン（黒）」をベースとし、強調部分だけ鮮やかな赤や青を使用
* ロゴはそのまま黒色で左上に配置
