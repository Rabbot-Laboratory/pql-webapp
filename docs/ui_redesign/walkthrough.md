# RABBOT LABORATORY - Cyber Dark UI 実装の完了

ご要望に応じて、最も映える「**案A: Cyber Dark（ダークモード）**」の方向性でデザインを刷新しました！アクセントには黒・白のロゴにぴったりな「ネオンシアン」を採用し、近未来のハイテク機材のような仕上がりになっています。

## 変更内容のサマリー

* **RABBOT LABORATORY ロゴの配置**
  * ヘッダーのテキストを削除し、専用のロゴ画像を `<img>` タグで配置しました。
  * 背景に合わせて白黒反転と輝度調整を行い、ダーク背景に白く光るように最適化しています。
* **全体カラーパレットの刷新**
  * ディープブラック（`#0b0c10`）をベースにし、パネルには半透明のダークグラス（ガラス風）スタイルを適用しました。
  * コントラストとなるアクセントカラーに「ネオンシアン（`#66fcf1`）」を設定しました。
* **UIシェイプのメカニカル化**
  * ボタンやパネルの角（Radius）を丸い形状から「シャープな四角形」に削り、工業用コンソールとしての堅牢さを演出しています。
* **3Dビューアのサイバー化**
  * 3Dロボットビュワーの背景を透過させてシステム全体と馴染ませました。
  * さらに、リムライト（逆光）やフォーカス時の発光色をシアンに変更し、ロボットもサイバーな照らされ方をするように調整しています。

## Verification
* `npm run build` にてコンポーネントおよび新アセット(`logo.png`)の依存関係とビルドが正常に終了することを確認しました。

> [!NOTE]
> ダークモードやシアンの配色、ネオンの強さなど、もう少し調整したい色味があればいつでもお申し付けください。展示会用により文字を太くするなどの対応も可能です。
